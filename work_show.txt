InfluxDB是一个开源的没有外部依赖的时间序列数据库。适用于记录度量，事件及执行分析。
https://github.com/influxdata/influxdb
https://portal.influxdata.com/downloads
mac安装
brew install influxdb
linux安装
wget https://dl.influxdata.com/influxdb/releases/influxdb-0.13.0.armhf.rpm
sudo yum localinstall influxdb-0.13.0.armhf.rpm

influxdb使用的端口如下：
http://www.361way.com/influxdb-user/5291.html
8083: Web admin管理服务的端口, http://localhost:8083
8086: HTTP API的端口
8088: 集群端口(目前还不是很清楚, 配置在全局的bind-address，默认不配置就是开启的)

Nginx 处理请求的过程一共划分为 11 个阶段，按照执行顺序依次是 post-read、server-rewrite、find-config、rewrite、post-rewrite、preaccess、access、post-access、try-files、content 以及 log.
